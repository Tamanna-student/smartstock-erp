# SmartStock ERP

Week 1 Assignment - MERN Stack Backend Setup

---

## Description

This project is the initial backend setup of a MERN Stack-based Inventory and Billing Management System.

---

## Completed Work

- Express.js server setup
- MongoDB Atlas connection
- Environment variable configuration
- Basic folder structure (MVC pattern)
- Security middleware setup (cors, helmet, morgan)
- Health check API

---

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- dotenv
- cors
- helmet
- morgan

---

## API Endpoint

- GET /api/health

---

## Setup Instructions

1. Clone repository  
2. Go to backend folder  
3. Run npm install  
4. Create .env file  
5. Add MongoDB URI  
6. Run npm run dev
